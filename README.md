# Copy URLs to Markdown

選択中のタブのURLをMarkdown形式でクリップボードにコピーするChrome拡張機能です。

## 機能

- 複数選択したタブのタイトルとURLを一括でMarkdownリスト形式に変換
- ワンクリックでクリップボードにコピー
- コピー完了時に拡張機能アイコンにチェックマークを表示

### 出力形式

```markdown
- [タブのタイトル](https://example.com/page1)
- [別のタブのタイトル](https://example.com/page2)
```

## インストール方法

1. このリポジトリをローカルにクローンまたはダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このリポジトリのフォルダを選択

## 使い方

1. Chromeで複数のタブを選択（Cmd+クリック / Ctrl+クリック）
2. 拡張機能のアイコンをクリック
3. クリップボードにMarkdown形式のリストがコピーされます
4. お好みのエディタに貼り付け

## 動作環境

- Chrome / Chromium系ブラウザ
- Manifest V3対応

## 権限

- `tabs`: タブ情報の取得
- `clipboardWrite`: クリップボードへの書き込み
- `scripting`: クリップボードアクセス用のスクリプト実行
- `activeTab`: アクティブなタブへのアクセス
